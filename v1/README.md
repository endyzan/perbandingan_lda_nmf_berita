# pertanyaan pengukuran
Lebih baik mana LDA atau NMF?
cara membuktikan nya dengan cara menghitung / melakukan pengukuran.
tiap model (LDA dan NMF) memiliki cara untuk mengukur nya.

nah cara untuk mengukur itu tuh di sebut "matrik".

jadi kita harus menggunakan matrik yang bisa di terapkan / di pakai pada keduanya.

misal:
matrik Coherence (Cv): LDA NMF bisa
matrik Topik Diversity (TD): LDA NMF bisa

matrik perplexity (PP): itu cuma bisa di pakai buat ngukur model seperti LDA
- alasan nya: pp itukan ngukur probabilistik
